# Examen-Final-Pensamiento-Computacional
Proyecto desarrollado para el Examen Final de Pensamiento Computacional. Consiste en un sistema interactivo programado en p5.js que utiliza estados, eventos, variables, condicionales, bucles, funciones, multimedia e interacción mediante mouse para generar una experiencia lúdica basada en la captura de burbujas y la obtención de puntaje.
